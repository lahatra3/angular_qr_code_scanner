# angular_qr_code_scanner

### Instruction:
```Shell
    $ git clone https://github.com/lahatra3/angular_qr_code_scanner.git
    $ cd angular_qr_code_scanner/qr_code_scanner
    $ npm install
    $ ng serve # optionnelle, pour le rendu web (http://localhost:4200)
    $ ng build qr_code_scanner
    $ npx cap run android # connecter votre android(connecter sur wifi) sur PC
```
